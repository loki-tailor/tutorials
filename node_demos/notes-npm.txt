major_version.minor_version.patch

8. major version : major changes: breaks the API
2. minor version : new features : does not break API
6. patch : bug fixes

"lodash": "^4.17.4" > when someone else does "npm install", it will install latest "minor version", but not install latest major version
"lodash": "`4.17.4" > when someone else does "npm install", it will update to latest "patch version", but not install latest major version or minor version
"lodash": "4.17.4" > when someone else does "npm install", it will install exact version
"lodash": "*" > when someone else does "npm install", it will install latest major.minor.patch versions
