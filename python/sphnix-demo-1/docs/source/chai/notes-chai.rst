CH 1: Introduction to Chai
-------------------------------

* Chai is an assertion library for node js and can be paired with any javascript framework
* Assertions are used to ensure the assumptions and the acctual results of the function are same as they should be.

* Chai Interfaces:
    - BDD style
        - Expect 
        - Should
    - TDD style
        - Assert
